# SAM
## Quick CheatSheet
- Create a virtual envvironment 
    - ```python3 -m venv venv```
    - ```source venv/bin/activate```
- Install the dependencies
    - ```pip install -r requirements.txt```
- Additional installations
    - ```pip install torch```
    - ```pip install nltk```
    - ```pip install -U g4f[all]```
- USED : ```pip freeze > requirements.txt```
